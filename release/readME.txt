Диапозоны:
	Ввод:
		Натуральное число (unsigned long long) - Диапозон 0 до 18446744073709551615.
		Целое число (long long int) - Диапозон −9223372036854775808 до +9223372036854775807.
		Степень полинома - Целое число.
	Вывод:
		Для каждого типпа данных результирующее число ограничивается только лишь входными данными.

Справка к полиномам(многочленам):
	При нажатии на кнопку выводится окно, в котором необходимо ввести степень старшего неизвестного. Далее вызываются окна:
окно числителя(целое число), окно знаменателя(натурально число). И так до тех пор пока не будут введины все коэффициенты неизвестных полинома.
	Пример:
	Если вы вводите степень p = 9, то 10 раз отобразятся окна с вводом числителя и знаменателя.
	1 вызов: Ввод коэффициента x^p. (окно числителя(целое число), окно знаменателя(натурально число))
	p+1 вызов: Ввод коэффициента x^0; (окно числителя(целое число), окно знаменателя(натурально число))
